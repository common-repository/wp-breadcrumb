== Description ==

Plugin creates advanced breadcrumb in wordpress pages.<br>
Плагин создает "хлебные крошки" - навигацию в виде цепочки. 

== Installation ==

=Installation=
1. Upload plugin files to the `/wp-content/plugins/` directory.<br>
2. Activate the plugin through the 'Plugins' menu in WordPress.<br>
3. Paste code in template
<pre><code class="php">&lt;?php wp_breadcrumb_show_full();?&gt;</code></pre>

=Установка=
1. Загрузите плагин в директорию: `/wp-content/plugins/` directory.<br>
2. Активируйте плагин через меню 'Plugins' в административной панели Wordpress.<br>
3. Вставьте код в шаблон Wordpress
<pre><code class="php">&lt;?php wp_breadcrumb_show_full();?&gt;</code></pre>
